Data Flow

Two Parallel Event Lifecycles

Early warnings (P-Wave alarms) bypass the write queue entirely — direct insert, sub-second delivery. Regular event data uses a 2-second batch for DB efficiency. Two separate paths, two different priorities.

Bypasses the 2s queue — direct await insert()

Authentication Architecture

Supabase Auth integrates with zero changes to the aggregator or Realtime pipeline. Only RLS policies are updated to require auth.role() = 'authenticated'.
